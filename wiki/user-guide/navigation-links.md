# Navegacion Interna

Guia de usuario para enlaces rapidos entre secciones.

## Dashboard

- **Balance Total**: toca la tarjeta para ver el listado de transacciones.
- **Invertido Trading**: toca la tarjeta para ir al listado de operaciones.
- **Ranking de simbolos**: toca una fila para abrir el detalle del simbolo.

## Detalle de operacion

- **Simbolo** (logo o nombre): toca para abrir el detalle del simbolo.
- **Precio actual**: toca el bloque para abrir el detalle del simbolo.

## Notas

- Los enlaces funcionan dentro de tu cuenta activa.
- Si no hay simbolo disponible, el enlace no se muestra.
