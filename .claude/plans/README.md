# Planes de Desarrollo

## Estructura
Cada plan consta de:
1. `{nombre}_PLAN.md` - Plan inmutable (solo lectura)
2. `{nombre}_TRACKING.md` - Seguimiento mutable

## Formato Plan (inmutable)
```markdown
# Plan: {Nombre}
Fecha: YYYY-MM-DD

## Objetivo
## Fases
## Archivos afectados
```

## Formato Tracking (mutable)
```markdown
# Tracking: {Nombre}

## Estado: En progreso | Completado | Pausado

## Fases
- [x] Fase 1 - Descripción breve
- [ ] Fase 2 - Descripción breve

## Notas
```
